- Default mode: self-read / leave-behind; requires summary-layer + appendix-layer
- Takeaway titles only — no label titles
- One screen = one cognitive job
- No decorative imagery — every visual carries cognitive function or is deleted
- Evidence integrity: key claims need source + context + explanatory text + methodology caveat
- Rebuild when thought/architecture/audience changes; revise only for local execution fixes
- Final deliverable must survive without a presenter
