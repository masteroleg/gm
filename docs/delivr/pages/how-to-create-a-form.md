How to create a form.
=====================

How to create a form.

Delivr allows you to easily create and manage forms using its **Modules** system. Attach forms to landing pages or profiles to gather user information, generate leads, enable signups, and more—all with minimal effort.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

How to create a form.

**To get started.**
-------------------

*   Click the **Modules** link in the top navigation bar
*   Then click **Create Form**

_Forms are reusable modules; you can attach them to multiple pages and profiles._

**Form Basics**
---------------

Begin by entering general information:

*   **Internal Title.** Used for your reference only (not visible to users)
*   **Public Title.**_(Optional)_. Displayed above the form
*   **Language.**Choose the language for field labels and system text
*   **Instructions**_. (Optional)_. Add instructions that appear above the form

**Add Fields**
--------------

Check the boxes to enable the fields you want to collect. You can include:

*   **Contact Fields:**
    *   Full Name
    *   First Name / Last Name
    *   Email
    *   Phone
    *   Company
    *   Address (Street, City, State/Prov, ZIP, Country)

*   **Optional Fields:**
    *   Promo Code
    *   Date of Birth
    *   Age Range (Custom text per range)
    *   Gender
    *   Message field

*   Add up to 3 consent or agreement checkboxes with custom text
*   Enable bot protection

**Submit Button & Terms**
-------------------------

*   Submit Button Label: Customize the button text (e.g., “Sign Up” or “Get Started”)
*   Button Color: Set the button background color using a HEX code
*   Terms Heading & Content: Display disclaimers or agreements below the button

**Form Behavior After Submission**
----------------------------------

You have four options for what happens after a user submits the form:

**Option 1**. Display a Success Message
---------------------------------------

*   Add a Title, Subtitle, and Message
*   Optionally add a Button with text and a URL (e.g., “Return to Website”)

**Option 2**. Display a Scratch Off
-----------------------------------

*   Choose from existing scratch-off modules (or create one)

Op**tion 3.** Display an Offer
------------------------------

*   Choose from existing offer modules (or create one)

**Option 4**. Redirect to a URL
-------------------------------

*   Automatically send users to a thank-you page, download link, or another website _Note: If a Redirect URL is set, it will override all other success options._

**Response Settings & Notifications**
-------------------------------------

*   Allow Users to Fill In Form Again: Choose how frequently the same user can submit (default is "Never"; note that users can resubmit if they clear browser cookies)
*   Notification Email: Enter one or more email addresses (comma-separated) to receive submissions

_Delivr does not store form responses on the platform. They are only sent via email._

**Save Form**
-------------

Once your form is complete, click the **SAVE & CLOSE** button.

Best Practices

*   Keep forms short to encourage completions
*   Use checkboxes for privacy or marketing consent
*   Set a redirect URL for downloads or next steps
*   Test your form before publishing it on a live page
