How to create and manage a GS1 Digital Link QR code.
====================================================

How to create and manage a GS1 Digital Link QR code.

GS1 Digital Link allows a single GS1-compliant QR code to connect packaging to product information, traceability data, and consumer experiences. With Delivr, you can create and manage GS1 Digital Links directly from your dashboard—keeping compliance, analytics, and routing in one place.

Delivr allows you to convert an existing campaign into a GS1-compliant Digital Link directly from your dashboard.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Steps to create an account.
---------------------------

➀

Access the GS1 Digital Link Option
----------------------------------

**After logging in:**

1.   Go to your **Dashboard and**create a new campaign
2.   Or alternatively, locate the existing campaign you want to use
3.   Under **Actions**, click the **GS1** blue button

This opens the **GS1 Digital Link management view** for that campaign.

➁

Set Up Your GS1-compliant Digital Link & Code
---------------------------------------------

Once inside the GS1 view, you can generate a GS1-compliant Digital Link using one of two options: **Simple** or **Extended**.

Simple (GTIN only)
------------------

Use this option when you only need a basic GS1 Digital Link.

*   **Global Trade Item Number (GTIN)**
    *   Identifier: /01/
    *   Example: 44444444444442
    *   Required for all GS1 Digital Links

This creates a compliant Digital Link and QR code using the GTIN alone.

Extended (GTIN + qualifiers + attributes)
-----------------------------------------

Use this option when you need product-level specificity or additional data.

Key Qualifiers (optional):
--------------------------

*   **Consumer Product Variant** /22/ 
    *   Up to 20 characters

*   **Batch or Lot Number** /10/ 
    *   Up to 20 characters

*   **Serial Number** /21/ 
    *   Up to 20 characters

Standard & Custom Attributes (optional):
----------------------------------------

*   Added as query parameters
*   Enter as a query string
*   Reserved characters must be percent-encoded

**Example:**

`?17=250101&style=6784563`

*   Up to 255 characters

Extended mode allows you to build a richer GS1 Digital Link while maintaining compliance.

➂

Generate and Download your Digital Link QR Code
-----------------------------------------------

Once saved, Delivr generates your **GS1 Digital Link URL and QR code**, for example:

`https://delivr.com/24sq2/01/4444444444444`

**You can download the QR code in multiple formats:**

*   **Vector formats** (for packaging and print)
*   **Raster formats** (for digital and general use)

Important usage notes.
----------------------

*   The **consumer-facing Destination URL** can be updated dynamically
*   Any changes to the**GTIN, key qualifiers, or attributes require generating and using a new QR code**
*   The **GTIN and key qualifiers** are passed to: 
    *   Delivr Analytics
    *   The Destination URL

*   **Standard and custom attributes are not passed** to analytics or the destination URL

Navigation options.
-------------------

**At the top-right of the GS1 Digital Link page, you’ll find:**

*   GS1 — view and manage all GS1 Digital Links
*   Dashboard — return to your campaign list

Alternatively, you can access all your GS1 Digital Link campaigns on the dashboard under the **Digital Links** tab.

Summary
-------

Delivr’s GS1 workflow lets you create and manage GS1-compliant Digital Links from any campaign—supporting simple GTIN use cases or advanced, attribute-rich implementations while keeping analytics and routing centralized
