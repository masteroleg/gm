# Awesome NotebookLM Prompts / Distillate

## What this is

- Curated public prompt collection for NotebookLM and adjacent slide-generation tools.
- Use in this project as a style-and-structure inspiration library, not as a direct source of truth.

## Why it matters here

- Confirms that strong NotebookLM outputs depend on explicit layout, hierarchy, language, and format constraints.
- Reinforces rules already validated in this project:
  - one slide = one message
  - strong hierarchy must be named, not assumed
  - exact language constraints must be explicit
  - diagram grammar must be specified

## Source signals captured

- Repository is organized around prompt families by visual style.
- Public style families visible in the README include:
  - Editorial / Business
  - Typography-led
  - Artistic / Avant-Garde
  - Professional / Product / Premium
  - Pop / Youth / Street
- Example style blocks visible in the README include:
  - Modern Newspaper
  - Sharp-edged Minimalism
  - Yellow x Black Editorial
  - Black x Orange Creative Agency
  - Neo-Retro Dev Deck

## Reusable patterns worth importing

- Explicit visible-copy rule:
  - ban markdown symbols and decorative text noise
- Explicit hierarchy rule:
  - massive headline
  - clear ratio between headline, subhead, and body
- Explicit layout rule:
  - asymmetry
  - grid
  - deliberate whitespace
- Explicit diagram rule:
  - problem vs solution
  - flow / formula / arrow-step diagrams

## Safe use in this project

- Use only as prompt inspiration for:
  - hierarchy
  - spacing
  - editorial structure
  - premium restraint
- Best-fit influences for this project:
  - Modern Newspaper
  - Sharp-edged Minimalism
  - premium studio / product polish

## Unsafe use in this project

- Do not import:
  - youth / street / manga / rebellious developer aesthetics
  - mixed-language headline systems
  - playful or ironic styling that weakens industrial credibility
  - decorative styles that overpower the sales narrative

## Non-negotiable local adaptation

- All outgoing buyer-facing artifacts in this project must remain Ukrainian only.
- Never let prompt-bible inspiration override:
  - `genu.mark first`
  - `strong core first`
  - external `ERP` and `eАкциз`
  - regulated industrial premium visual language

## How the next agent should use this reference

- Use it to improve prompts and visual rules.
- Do not upload it into NotebookLM as a regulated generation source.
- Do not treat it as permission to loosen current narrative or language policy.
